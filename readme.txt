== Description ==

FBChat FREE renders a facebook like chat for your Wordpress site allowing you to chat with logged in users
You can configure details settings for your plugin in admin area, see manual for more informations
You can download the +PLUS version at this link http://www.2punti.eu/wpfbchatplus.html

== Installation ==

Upload the FBChat FREE plugin to your blog, Activate it, and you'll see a bottom fixed bar on frontend site
that allow you chatting with other users 
You can configure details settings for your plugin in admin area, see manual for more informations
You can download manual at this link http://www.2punti.eu/wp/wpfbchatfree.pdf
